# Wave 3: Validation, Trust and Scope Convergence

## Findings

- Trust/export/delete/simple pricing are table stakes, not a standalone wedge. Competitors already offer strong export, account deletion, sync and one-time pricing in various combinations.
- The evidence-ranked MVP is a private, low-friction memory system: scan/edit/confirm, would-buy-again, search/filter, durable sync/delete/export, and manual fallback. Sharing is optional; passport is progressive.
- Recommended validation sequence is gated: Korean-bag benchmark -> capture-method comparison -> positioning test -> third-distinct-bag retention -> real payment commitment.
- Stronger positioning is “좋았던 원두를 잊지 않고 다시 찾는 가장 빠른 방법.” Beauty is the experience layer; passport is an earned retention layer.
