# 🪵 CoffeeDex Premium Wood Cabinet & AI Quick Tuning Design

이 문서는 사용자의 동의를 얻어 확정된 **프리미엄 우드 캐비닛 UI** 및 **원클릭 AI 바리스타 퀵 튜닝 루프**의 최종 상세 디자인 규격입니다.

---

## 1. 우드 캐비닛 진열장 UI (Section 1)

원클릭 및 개인 수집 감성을 극대화하기 위해 기존의 평범한 그리드뷰 대신 홈카페 감성의 다크 우드 진열장을 시각화합니다.

### 🎨 비주얼 레이아웃 및 스타일 가이드
*   **찬장 배경(Cabinet Shelf)**:
    - 찬장 느낌의 가로선과 입체적인 다크 우드 질감 플레이트 설계.
    - 배경색: 트루 블랙 (`#000000`) 및 에스프레소 브라운 그라디언트 (`bg-gradient-to-b from-[#15100c] to-[#0d0907]`).
    - 각 열의 하단에 두께감 있는 선반 라인 플레이트 렌더링 (`border-b-4 border-[#2d1f16] shadow-[inset_0_4px_6px_rgba(0,0,0,0.9)]`).
*   **원두 에이징 라이트(Ambient Glow Status)**:
    - 원두의 로스팅일 및 개봉일 기준 신선도 연산 결과(`evaluateFreshShelfStatus`)에 따라 원두 패키지 카드 테두리에 은은한 글로우 효과 부여.
    1.  **디개싱 완료 대기 (Glowing Gold)**: 로스팅 7일 이내. 탄산가스가 방출 중인 풋풋한 단계.
        - 테두리: `border-yellow-500/40 shadow-[0_0_15px_rgba(234,179,8,0.2)]`
    2.  **맛의 피크 타임 (Amber Breath)**: 로스팅 7일~30일 사이이면서 개봉 14일 이내. 최상의 맛.
        - 테두리: `border-[#C58948]/50 shadow-[0_0_15px_rgba(197,137,72,0.25)]`
    3.  **맛 손실 진행 / 에이징 만료 (Dimmed Gray)**: 개봉 30일 초과 또는 로스팅 45일 초과. 빠른 소비 요망.
        - 테두리: `border-white/10 opacity-70`

## 2. 원클릭 AI 바리스타 퀵 튜닝 (Section 2)

원두 상세 뷰에서 사용자의 직관적인 맛 피드백 한 번으로 다음 추출을 보정해 주는 초간편 피드백 가이드입니다.

### 🔄 상호작용 및 API 설계
*   **피드백 옵션 (원클릭 버튼)**:
    1.  `너무 신맛이 강해요 🍋` (too_sour)
    2.  `너무 묵직하거나 써요 ☕` (too_bitter)
    3.  `싱겁고 연해요 💧` (too_watery)
    4.  `딱 맛있고 좋아요! 💛` (perfect)
*   **AI 바리스타 API 연동 (`POST /api/v1/ai-barista`)**:
    - **요청 Body 규격**:
      ```json
      {
        "beanId": "uuid-string-of-shelf-item",
        "feedback": "too_sour"
      }
      ```
    - **바리스타 처방전 (Tuning Output)**:
      - API는 해당 원두의 로스팅 포인트 및 현재 가공 방식 정보를 활용하여 보정 레시피 가이드라인 텍스트를 반환합니다.
      - *예 (너무 쓴맛 대응)*: "현재 원두가 다크 로스팅 상태이며 과다 추출 경향을 보입니다. 다음 브루잉 시 **물 온도를 89°C로 낮추고**, **분쇄도를 한 단계 크게** 조정하여 쓴맛 추출을 제안해 드립니다."
*   **일지 저장 연동**:
    - 피드백 가이드 카드 하단에 `[📥 보정된 레시피 일지에 저장]` 버튼 노출.
    - 클릭 시 `POST /api/v1/brewing-logs` API를 호출하여 추천된 보정 레시피 정보(물 온도, 분쇄 크기 등)가 포함된 일지를 자동 저장합니다.
