# CoffeeDex Premium Mobile Redesign Research

Date: 2026-06-29

## Research Summary

- Coffee record apps such as iBrewCoffee and local Coffee Log prove demand for bean, brew, rating, photo, and searchable memory capture, but their value is strongest after the user can quickly recover a past brew or product.
- Blue Bottle and Fritz show that coffee purchasing is increasingly subscription and cadence driven, so CoffeeDex should make rebuy timing visible without pretending to be a marketplace.
- Korean specialty brands such as Fritz and Terarosa lean on tactile packaging, warm materials, and place-based brand memory rather than cold analytics language.
- Coffee communities repeatedly surface a simple need: remember what was brewed, what worked, what failed, and what to buy again. Community features are noisy unless the private memory loop is already strong.
- SaaS dashboard references are useful for hierarchy and status panels, but CoffeeDex should avoid a generic analytics cockpit. It should feel like a personal coffee room with operational panels.

## Chosen Direction

CoffeeDex now uses a "ceramic coffee room" direction: warm canvas, paper/linen surfaces, espresso text, clay action color, and dark espresso panels only for high-value operational loops.

This direction keeps the current product positioning:

- private coffee memory
- owned bean shelf
- repurchase recall
- taste passport
- Rebuy Intelligence
- Dial-in Coach

It intentionally avoids:

- marketplace or partner-offer claims
- public community feed styling
- one-note black gallery UI
- generic purple/blue SaaS gradients

## UI Principles

- First screen should answer: what do I own, what should I drink, what should I buy again?
- Cards and shelf items should feel tactile and private, not like admin widgets.
- Rebuy Intelligence and Dial-in Coach are premium operation panels, so they keep an espresso surface with high contrast.
- Mobile WebView should read as an app: safe spacing, bottom navigation, short headings, no horizontal overflow, and visible touch targets.
- Korean copy should remain personal and practical.
